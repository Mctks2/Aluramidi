# Projeto Alura Midi

O Alura Midi é um instrumento musical de interface digital, onde cada botão reproduz o som de um instrumento.

Curso de JavaScript para Web: Crie páginas dinâmicas


## 🛠️ Tecnologias utilizadas
- HTML
- CSS
- JS
